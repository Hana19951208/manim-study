# /quiz: 每日练习生成器

本命令用于根据当前学习天数的 `README.md` 内容，自动生成配套的练习题 (`exercise.py`) 和标准答案 (`solution.py`)。

## 执行步骤

1. **确定当前天数**：识别当前正在学习的文件夹（例如 `week01_basics/day02_text/`）。
2. **分析知识点**：
   - 读取该目录下的 `README.md` 中的“学习目标”和“核心知识点”。
   - 提取关键的 API（例如 `MathTex`, `Transform` 等）。
3. **生成练习模板**：
   - 在该目录下创建 `exercise.py`。
   - 设计 4-5 个由浅入深的任务，其中必须包含 1-2 道具有挑战性的**进阶思考题**（涉及多种 API 组合或复杂排版）。
   - 每个任务以详细的中文字符串注释（Docstring）描述。
   - **强制规范**：在 `if __name__ == "__main__":` 的 `tempconfig` 中，必须包含 `"input_file": __file__`。
4. **生成参考答案**：
   - 在该目录下创建 `solution.py`。
   - 实现上述任务的标准代码，并添加详尽的中文注释解释关键逻辑。
5. **通知用户**：告知练习已生成，并提供渲染预览命令。

## 示例触发
用户发送：“我学完今天的知识了，出几道题考考我”、“题目太简单了，再来几道难的”或直接输入 `/quiz`。
